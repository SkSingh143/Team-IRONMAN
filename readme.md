# LiveCollab 🚀

Real-time collaborative whiteboard with voice chat, code sharing, and live polling.

---

## Features

- Real-time drawing (WebSockets)
- Live cursor tracking
- Voice chat (WebRTC)
- Code sharing panel
- Live polling
- Room-based collaboration
- Admin controls (kick, clear)
- Reconnect & state recovery
- JWT authentication (access + refresh tokens)

---

## Tech Stack

**Frontend**
- React (Vite)
- Canvas API
- Zustand
- WebSocket

**Backend**
- Node.js
- Express.js
- ws

**Database**
- MongoDB (Mongoose)

**Other**
- JWT Auth
- WebRTC
